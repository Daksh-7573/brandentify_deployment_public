You are a senior frontend engineer working on a portfolio builder inside a product called Brandentifier.

Goal:
Create a reusable React + TypeScript + Tailwind CSS component for a “Creative Quantum Portfolio” layout. It should look and feel like an expanded version of a Creative Quantum Card, using the same dark tech style, neon glows, gradients, and glassmorphism.

Tech requirements:
- React functional components (TypeScript)
- Tailwind CSS utility classes for styling
- Framer Motion optional for simple enter/hover animations (but keep it light)
- No external UI framework (no MUI, no Chakra, etc.)
- Code should be clean, composable, and production-ready

Component API:
Create a main component:

  interface Skill {
    id: string;
    name: string;
    proficiency: number; // 1-5
    category?: string;
    yearsOfExperience?: number;
  }

  interface Experience {
    id: string;
    title: string;
    company: string;
    industry: string;
    domain: string;
    location: string;
    startDate: string; // YYYY-MM
    endDate?: string | null;
    description?: string;
    keyResponsibilities?: string[];
  }

  interface Education {
    id: string;
    degree: string;
    institution: string;
    location?: string;
    fieldOfStudy?: string;
    industry?: string;
    startDate: string;
    endDate?: string | null;
    skillsAcquired?: string[];
  }

  interface Project {
    id: string;
    title: string;
    description: string;
    category: string;
    technologies: string[];
    outcome?: string;
    startDate?: string;
    endDate?: string | null;
    mediaUrls?: string[];
    externalUrl?: string;
    impact?: string;
    role?: string;
    teamSize?: number;
  }

  interface Service {
    id: string;
    title: string;
    description: string;
    category: string;
    priceInr?: number;
    priceUsd?: number;
    isHourly?: boolean;
    features?: string[];
    imageUrl?: string;
    order?: number;
    isActive: boolean;
  }

  interface CreativeQuantumPortfolioProps {
    // SECTION 1: HEADER
    name: string;
    photoURL: string;
    title: string;
    tagline?: string;
    location?: string;

    // SECTION 2: ABOUT / SNAPSHOT
    aboutMe: string;
    visionStatement?: string;

    uniqueValueProposition?: string;
    coreValues?: string[];
    primaryAudience?: string[];
    secondaryAudience?: string[];
    industry?: string;
    domain?: string;
    lookingFor?: string;
    email?: string;

    // SECTION 3: SKILLS
    skills?: Skill[];

    // SECTION 4: EXPERIENCE
    experiences?: Experience[];

    // SECTION 5: EDUCATION
    educations?: Education[];

    // SECTION 6: PROJECTS
    projects?: Project[];

    // SECTION 7: SERVICES (PREMIUM)
    isPremium?: boolean;
    services?: Service[];
  }

The component should be:

  const CreativeQuantumPortfolio: React.FC<CreativeQuantumPortfolioProps> = (props) => { ... }

Design system (use ONLY these colors):

Primary:
- Cyan: #22d3ee  (primary accent, glows, CTAs)
- Purple: #a78bfa (secondary accent)
- Blue: #3b82f6  (grid lines, accents)

Backgrounds:
- Dark Navy: #0A0F2C   (page background)
- Dark Purple: #1F1B44 (section gradients)
- Dark Blue Card: #0A1A3F (card backgrounds)

Text:
- Primary: #ffffff
- Secondary: rgba(255,255,255,0.9)
- Tertiary: rgba(255,255,255,0.6)

Glow & shadow:
- Use box-shadows like:
  - 0 25px 80px rgba(34,211,238,0.15)
  - 0 10px 40px rgba(167,139,251,0.1)
  - 0 0 20px rgba(34,211,238,0.3)

Global layout:
- Page background: full-screen gradient from Dark Navy to Dark Purple with a subtle grid overlay
- Max width: max-w-6xl mx-auto px-4 sm:px-6 lg:px-8
- Vertical spacing between sections: py-12 sm:py-16 lg:py-20

       - Card: “About Me” with full bio

   - Layout:

       - Title
       - Category
       - Price (INR and USD, with “/hour” or “per project” based on isHourly)

       - CTA button “Let’s Collaborate”
     - Cards should feel more “pricing table” premium with stronger glow and gradients

Styling rules:
- Use Tailwind classes wherever possible
- Reuse the Creative Quantum card style:

  - border border-white/10

Accessibility & responsiveness:
- Must look good on mobile, tablet and desktop
- Mobile: stack sections, full-width cards
- Desktop: use 2–4 columns where appropriate
- Ensure text is readable, buttons have good tap targets

What to output:
- The full React component file (with all subcomponents defined in the same file or as small internal functions)
- Use TypeScript typing for props
- Use Tailwind classes directly on elements
- Include basic modal implementation for project details (no external modal lib)
- No lorem ipsum: use sensible placeholder content logic if needed
