1. Visual Theme & Core Principles
Property	Setting
Theme	Dark
UI Style	Soft UI + Glassmorphism + Spatial UI
Transparency Level	Medium (40–50%)
Blur Strength	Medium (10–15px Gaussian blur)
Noise Texture	Subtle, layered at 10–15% opacity
Shadows	Soft glow, light drop shadows
Corners	Extra rounded (border-radius: 24px)
Ambient Background	Subtle spatial tone with ambient dark blur

📱 9. Mobile Responsiveness
Nav bar becomes hamburger menu.

Cards full-width with vertical stack.

Forms and inputs: 100% width, stacked vertically.

Use motion-safe animations only on mobile to avoid lag.

🛠️ 10. Tools & Frameworks Suggestions
TailwindCSS with backdrop-filter, blur, and dark mode utilities

Framer Motion or GSAP for animations

SVG Noise Overlay or backdrop-image CSS noise texture
