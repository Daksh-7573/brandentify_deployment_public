4️⃣ ANIMATION SYSTEM — NEW & UNIQUE
Keyframes
1. inkPulse

Ink grows & fades like liquid.

@keyframes inkPulse {
  0% { transform: scale(0.95); opacity: 0.6; }
  50% { transform: scale(1.05); opacity: 1; }
  100% { transform: scale(0.95); opacity: 0.6; }
}


Used for:

Behind profile pic

Section dividers

2. gridMove

Slight parallax movement of grid.

@keyframes gridMove {
  0% { background-position: 0 0; }
  100% { background-position: 20px 20px; }
}

3. posterReveal

Poster folds open.

4. duotoneHover

Changes image to cyan/magenta duotone on hover.

5. shapeMorph

Profile pic background smoothly changes between blob shapes.
