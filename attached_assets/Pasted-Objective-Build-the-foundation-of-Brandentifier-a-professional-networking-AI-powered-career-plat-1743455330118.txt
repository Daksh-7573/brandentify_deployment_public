Objective:
Build the foundation of Brandentifier, a professional networking & AI-powered career platform, starting with a web version. The initial phase includes:
✅ UI/UX design for intuitive user experience
✅ Frontend (Next.js with Tailwind CSS)
✅ Backend (Node.js, Express.js, Firebase/MongoDB)
✅ Authentication (OAuth with Google Login via Firebase/NextAuth)
✅ Scalable Database Architecture

1️⃣ UI/UX Design Guidelines
User Flow & Pages:
Landing Page (Before Login)

Hero section with Brandentifier’s unique value proposition

CTA: “Sign Up with Google”

Key benefits in a visually engaging layout

Dashboard (After Login)

Left Sidebar: Navigation (Profile, AI Career Booster, Industry Feed, Jobs, Settings)

Center Feed: Personalized AI-curated career content

Right Panel: Profile snapshot, resume completion progress, notifications

Profile Page

Upload Resume (PDF/Word)

Profile Sections: Work Experience, Skills, Education

AI Resume Score & Keyword Optimization

AI Career Booster (Mark - AI Chatbot)

Chat-style interface for career guidance

AI-generated career path suggestions

Personalized learning recommendations

Job & Skill Suggestions Page

AI-based job matching & skill gap analysis

Course recommendations (LinkedIn, Udemy, Coursera APIs)

2️⃣ Tech Stack (Best for Scalability & Performance)
✅ Frontend:

Next.js (React framework for SSR & better SEO)

Tailwind CSS (for a clean, modern UI)

ShadCN / Material UI for UI components

✅ Backend:

Node.js with Express.js (Fast & scalable API development)

Firebase for authentication (Google Sign-In)

REST API structure with future GraphQL support

✅ Database:

MongoDB (via Mongoose) → Flexible & scalable for storing user data, profiles, job listings

Firebase Firestore (for real-time updates if needed)

✅ Authentication:

Google OAuth via Firebase / NextAuth

JWT-based authentication for secure user sessions

✅ Deployment:

Vercel (for frontend hosting - best with Next.js)

AWS Lambda or DigitalOcean (for backend API hosting)

MongoDB Atlas (managed cloud DB for scalability)
