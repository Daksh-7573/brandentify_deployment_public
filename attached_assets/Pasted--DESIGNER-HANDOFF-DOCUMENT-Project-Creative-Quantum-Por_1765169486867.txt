🎨 DESIGNER HANDOFF DOCUMENT
Project: Creative Quantum Portfolio
Base System: Creative Quantum Card (you shared)
Use: Figma / UI Design
1. Design Goal

Create a single-page portfolio layout that feels like an expanded version of the Creative Quantum Card:

Same dark tech aesthetic

Same cyan/purple/blue palette

Same glassmorphism + grid + SVG chips/circuits

Same animations and hover behaviors, but applied to full sections

Works for: designers, developers, marketers, creators

Tone:
Premium, futuristic, clean, professional.

Avoid:

Overloaded neon gamer style

Crowded text blocks

Too many fonts or random colors

2. Global Visual System
Colors (use ONLY from card system)

Primary:

Cyan: #22d3ee

Purple: #a78bfa

Blue: #3b82f6

Violet: #8B5CF6

Backgrounds:

Dark Navy: #0A0F2C

Deep Purple: #1F1B44

Dark Blue Card: #0A1A3F

Text:

Primary: #ffffff

Secondary: white @ 90%

Tertiary: white @ 60%

Borders & Glow: reuse Cyan/Blue/Purple border & shadow system exactly from the card.

Typography

Use the same font stack as Creative Quantum Card (Tailwind default sans/Inter-style):

Name / Major headings

Weight: bold

Size: 24–32px desktop

May use soft gradient text occasionally (not everywhere)

Section titles

Size: 18–22px

Weight: semibold

Color: white / cyan accent

Body / paragraphs

Size: 14–16px

Weight: regular/light

Color: white/90

3. Global Layout & Background

For the whole page:

Background gradient:
linear-gradient(135deg, #0A0F2C, #1F1B44)

Grid overlay:
Light SVG grid pattern (40x40), opacity ~20%

Corner decorations (per major section, not necessarily every card):

Top-right: circuit SVG (same style as card)

Bottom-left: chip/grid SVG

Opacity around 15–25%

Use one decorative SVG per section, not all over the place — keep it tasteful.

Spacing:

Section padding: 80px top/bottom, 24px left/right

Section max width: 1200px centered

5. Animations (Implementation)

Use Tailwind keyframes defined for card:

animate-pulse-slow for profile glow

animate-ping for active dot

animate-pulse for loading skeletons

For scroll/entrance animations, you can:

Use Framer Motion or

Use CSS with opacity-0 translate-y-4 → opacity-100 translate-y-0 when section is visible via IntersectionObserver (optional).

On mobile:

7. Performance Notes

Lazy load images (loading="lazy")

Use optimized thumbnails for project media

Don’t animate layout-affecting properties (top, left); use transform & opacity

8. Accessibility

Make CTAs button or a tags with aria-label

Modals must trap focus, provide close button

Respect prefers-reduced-motion (disable pulse/ping where possible)
