GENERAL UI DESIGN GUIDELINES (Inspired by Vision Pro, for Web Use)
1. Modern Minimalism with Depth
Use a clean, soft, minimalist design with depth to create visual hierarchy.

Leverage glassmorphism principles:

Semi-transparent cards

Blur effects (CSS backdrop-filter)

Soft drop shadows for elevation

✅ Use for dashboards, content previews, side panels, and modals.

2. Soft Rounded UI Elements
All containers and cards should have rounded corners (16–32px).

Buttons, thumbnails, and media previews follow the same rounded aesthetic.

✅ Creates a calm, premium look aligned with futuristic interfaces.

3. Layered Hierarchy
Even though your layout is flat:

Simulate depth using layered sections:

Main content area

Secondary panels (sidebar, now playing, queue)

Utility layers (notifications, media controls)

✅ Use z-index + soft shadows + blur to imply depth without floating windows.

4. Large, Comfortable Click Targets
Design for touch + mouse:

Buttons: Min height 44px

Cards: Clickable areas should be clear and well-separated

✅ Inspired by Vision Pro’s gesture readiness — now applied for accessibility.

5. Visual Focus with Light and Blur
Use light gradients, ambient shadows, and blur to guide attention.

Make primary CTAs (e.g., “Start,” “Watch,” “Edit”) stand out with color, glow, or subtle animation.

✅ Create visual flow without hard contrast.

6. Sticky and Anchored Layouts
Since user can’t move the UI:

Use anchored sections: a fixed sidebar, sticky playback panel, collapsible side drawer.

Ensure responsive behavior on mobile: vertical stacking + expandable controls

✅ Keeps the layout intuitive and clean on all screen sizes.

7. Modular Grid System
Organize content (e.g., playlists, media cards) in responsive grids:

Cards with preview images

Titles, subtexts, and status badges

Optional icons (e.g., bookmark, like, play)

✅ Ensures scalability and reuse across the portal.

8. Typography & Icons
Font: Prefer San Francisco, Inter, or Manrope for modern look.

Icons: Use line icons (Lucide, Feather, or SF Symbols equivalent)

Font Sizes:

Headline: 24–32px

Subhead: 18–20px

Body: 14–16px

✅ Consistent type scale maintains professionalism.

9. Media Controls Design
Sticky mini player with:

Thumbnail + title

Play/Pause + Seek bar

Mute, Volume, Fullscreen

(Optional) Collapse/Expand mode

✅ Keep controls intuitive and persistent across screens.

10. Color Palette
Use a calm, neutral base with accent colors:

Background: Soft grey or gradient white

Cards: White or translucent light background

Accent: Deep blue, teal, or purple for interaction states

✅ Reflects elegance while being non-distracting for professional users.

📱 RESPONSIVE DESIGN STRATEGY
Breakpoint	Layout Behavior
Mobile (<768px)	Stack cards, collapse side panels into tabs or drawers
Tablet (768–1024px)	Show 2–3 column grid, sticky player at bottom
Desktop (>1024px)	Full grid, sticky side panel/player, sidebar always visible
