DEVELOPER HANDOFF DOCUMENT
Project: Holographic 3D Portfolio Template
System: Brandentifier
Platform: Web (React, Tailwind, Framer Motion recommended)
1. PROJECT GOALS

Build a high-performance portfolio UI with:

Holographic visuals

3D interactions

Smooth animations

Robust data mapping

Full responsiveness

Accessibility compliance

All sections must be dynamic and populated from user data.

2. TECHNOLOGY STACK (RECOMMENDED)
Layer	Tech
Framework	React (TS)
UI	Tailwind + CSS Variables
Animation	Framer Motion
3D & Parallax	Vanilla JS / React-Spring
Icons	Lucide-React
Layout	CSS Grid + Flex
State	Props + Local derived state

Do not use heavy 3D libraries (Three.js) — performance costs too high.

7. ANIMATION SPECS

Use Framer Motion

7.1 Entrance Animations

Fade-Up

initial={{ y: 20, opacity: 0 }}
whileInView={{ y: 0, opacity: 1 }}
transition={{ delay: i * 0.1, duration: 0.4 }}

7.2 Hover Card
transition: 0.25s ease;
transform: translateY(-4px) scale(1.02);
box-shadow: var(--shadow-hover);

7.3 Parallax Glow (Mouse Track)
onMouseMove: (e) => {
  const x = (e.pageX / width) * 100;
  const y = (e.pageY / height) * 100;

  element.style.background = 
    `radial-gradient(circle at ${x}% ${y}%, rgba(34,211,238,0.25), transparent 80%)`;
}

7.4 Particle System (Background)

Create 15–20 random dots

Individual float animation

Random delay + duration

Don’t animate with JS — use CSS keyframes

8. RESPONSIVENESS

Mobile Rules:

Disable parallax

Scale down glow

Reduce animation intensity

9. ACCESSIBILITY

Respect prefers-reduced-motion

Keyboard accessible modals

Text contrast AA minimum

Semantic HTML structure

10. PERFORMANCE STRATEGY
Required:

Lazy load images

Lazy mount heavy components

Compress media

GPU accelerated animations

Prefer:

transform, opacity


Avoid:

top, left, width

11. API & DATA HANDLING
Data Source:

Passed via props or headless API

Rendering strategy:

Render lists as keyed maps

Pagination not required

Fallbacks for missing data

Example:

skills?.map(skill => (
  <SkillCard key={skill.id} {...skill} />
))
