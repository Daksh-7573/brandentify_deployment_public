🎨 1. VISUAL DESIGN & STYLING SYSTEM
🔹 Overall Theme

Editorial, minimalist, modern.

Large whitespace, high readability.

Coral highlight color to create emotional warmth and creative energy.

Card-based layout but VERY clean and non-techy.

🔹 Color System

Primary Colors

Purpose	Color	HEX
Background	Paper White	#FFFFFF
Text Primary	Rich Black	#0C0C0C
Text Secondary	Cool Gray	#575757
Accent	Coral	#FF6B5A
Soft Background	Porcelain	#F4F6F8
Line Color	Mist Gray	#E5E7EB
Premium Accent	Champagne Gold	#E2B45F
Color Usage Rules

Coral is used ONLY for:

CTAs

Active state chips

Small accent lines (left border)

Hover animations

Gold is used for:

Awards

Premium badge

Verified profile indicator

Background remains white for a clean editorial aesthetic.

✍️ 2. TYPOGRAPHY SYSTEM
Heading Font

GT America (Bold) or Satoshi Bold
→ For modern editorial look
→ Used for: Name, Section Titles, Service Titles

Body Font

Inter
→ High readability
→ Used for: About, Descriptions, Case Studies

Optional Accent

Playfair Display
→ Only for quotes or hero emphasis line
→ Very limited use

🌬️ 3. ANIMATION STYLE GUIDELINES

This template uses light, refined animations:

Fade Up (+10px) for all sections when entering viewport

Project Card Hover

Slight lift: translateY(-6px)

Subtle shadow: rgba(0,0,0,0.08)

Image contrast +3%

CTA Buttons

Scale 1.03 on hover

Coral glow shadow (soft)

Hero Title Reveal

Split text (each word fades in sequentially)

Delay: 60ms between words

Zero Neon / Zero Hacker Effects
→ Clean, luxury, editorial only.
