. Navigation Components
🧭 Top Navigation Bar
Placement: Fixed at the top or sticky scroll.

Style: Semi-transparent (glass effect), subtle shadow.

Contents:

Logo

Search Bar

Notifications Icon

Message/Inbox Icon

Profile dropdown

Responsive: Collapses to hamburger menu on mobile.

📚 Sidebar (Left Navigation)
Style: Vertical layout, glassmorphic panel, icons + labels.

Contents:

Home

Explore/Discover

Reels/Shorts

Bookmarks

Groups/Communities

Settings

🟣 2. Content & Feed Components
🧾 Post Card
Includes:

User info (avatar, name, timestamp)

Media (image/video carousel)

Caption

Tags/hashtags

Reactions (Like, Comment, Share, Bookmark)

Style: Rounded corners, soft shadows, max width.

Actions: Hover effects on buttons, expandable caption, modal for media.

🎥 Reels/Shorts Component
Vertical Video Preview with overlays:

Bottom-left: user info, caption

Right: like/comment/share vertical icons

Scroll behavior: swipeable on mobile, scroll on desktop

🧵 Thread/Discussion Panel
Used for: Replies, comment chains, forum-style posts.

Style: Indented replies, soft gray background, collapsible threads.

🔸 3. User Interaction Elements
🗨️ Comment Box
Inline or modal

Includes: Rich text, emoji, GIF picker, @mention

Auto-expand on typing, post on Enter, shift+enter for newline

💬 Chat/Messenger
Layout: Panel or popup

Features: Typing indicator, status (online/offline), read receipts, reactions

Extras: Optional voice message, attachments

💖 Reactions & Likes
Hover to expand emoji reactions

Like animation: subtle pop/shine on click

Count: rounded badges, adaptive formatting (e.g., 1.2K)

🟢 4. Media & Display Components
🖼️ Image/Video Viewer
Modal view with dark backdrop, swipe navigation

Zoom, fullscreen, download, and share options

🎧 Audio Player (Inline + Bottom Bar)
Inline: For posts or audio cards

Sticky Bottom Player: For podcasts/music

Includes: Play/pause, scrubber, volume, duration, cover art

📁 Carousel / Gallery
For multi-image or feature slides

Controls: Swipe, arrows, dots

Adaptive Layout: Horizontal scroll on mobile

🔵 5. Profile & User Interface
👤 Profile Card
Contents: Avatar, name, bio, stats (posts, followers, following), follow button

Style: Rounded, elevated, optional background banner

Mini variant: Used in mentions or sidebars

🔽 User Dropdown Menu
Profile

Settings

Notifications

Theme toggle

Logout

🏷️ Badge / Status
Used for: Verified, Admin, New user, Premium

Style: Small colored labels (rounded, filled or outlined)

🟡 6. Search & Discovery Components
🔍 Search Bar
Rounded input with icon, filters, auto-suggestions

Live results dropdown: categories (people, posts, hashtags, places)

🌍 Explore Feed / Discover Page
Grid or Masonry Layout

Filter tabs: Trending, People, Tags, Topics

Infinite scroll with sticky filter bar

🟤 7. UI Forms & Input
✍️ Create Post Modal
Rich text input with media upload

Tags, privacy setting (Public/Friends/Private)

Drag & drop media

📑 Forms & Inputs
Styled with soft blur or card background

Floating labels or pill-style inputs

Includes:

Toggle switches

Radio buttons

Multiselect with chips

Date/time picker

Sliders (volume, brightness, etc.)

🟠 8. System Components
🪧 Modal / Pop-up
Glass style with shadowed background

Used for: settings, sharing, confirmation

Close: Esc key, outside click

📣 Notifications
Top-right or side panel

Toast: For small, auto-dismissing messages

List view: Read/unread, grouped by type (like, comment, DM)

📊 Analytics / Insights Panel
Used for creators or community admins

Graph cards, follower growth, engagement metrics

Styled with glass chart overlays and dropdown filters

⚫ 9. Onboarding & Miscellaneous
🎬 Walkthrough / Intro
Multi-step guide with blurred overlay

Animated pointers or GIFs

Skip or "Got it" buttons

🎯 Empty States
Icons + small message (e.g., "No posts yet")

Suggest actions or tips

🔄 Loading Skeletons
Used while data loads

Mimics structure of posts/cards with shimmer animation

🔁 10. Responsive Behavior Guidelines
Screen Size	Layout Behavior
<768px (Mobile)	Stack all components, bottom nav, collapsible drawer
768–1024px	Show partial sidebar, scrollable main content
1024+ (Desktop)	Full layout: left nav, center feed, right panel

🧩 Design Tokens (Reusable Standards)
Token	Value Example
Primary Color	#6B73FF (Gradient: #6B73FF → #000DFF)
Background	rgba(255,255,255,0.7) (with backdrop blur)
Text Primary	#1C1C1E (dark), #FFFFFF (light mode)
Border Radius	16px for cards, 999px for buttons/tags
Spacing Unit	8px grid system (8, 16, 24, 32...)
Shadow	0 8px 16px rgba(0,0,0,0.05) (soft layered)
