Industry Pulse Logic Overview
Industry Pulse is a content feed system that displays professional posts called "pulses" from users across different industries. Here's the core logic:

1. Content Types
User Pulses: Regular posts from platform users
Project Pulses: Posts linked to specific projects
News Pulses: Industry news and insights
Musk Pulses: AI-generated content from the Musk AI assistant (user ID 3)
2. Smart Feed System
The feed uses multiple filtering and sorting mechanisms:

Tab-based Filtering:

"All": Shows all pulse types
"Musk News": Shows only AI-generated news pulses from Musk AI
Custom filters: Can filter by pulse type or industry
Smart Sorting:

Relevance-based: Calculates scores based on user preferences, industry matches, and engagement
Recency-based: Standard chronological sorting by creation date
Users can toggle between these sorting modes
3. Automated Content Generation
The system includes an intelligent pulse automation system:

Scheduled Generation:

Morning pulses (9 AM): Career insights and industry updates
Afternoon pulses (2 PM): Professional development and market analysis
Evening pulses (7 PM): Networking opportunities and skill development
Event-Driven Generation:

Monitors industry activity spikes
Automatically generates targeted content when specific industries show high activity
Uses AI to create relevant, timely content based on current events
4. Content Intelligence
User Context Analysis:

Analyzes user industries, domains, and locations
Generates personalized content based on community demographics
Targets specific user groups with relevant insights
Project Pulse Caching:

Pre-fetches project data for faster loading
Maintains global cache for project-related pulses
Optimizes performance for content with linked projects
5. Real-time Features
Smart Refresh System:

Auto-refresh every 60 seconds
Detects new premium content
Shows refresh indicators when new content is available
Toast notifications for feed updates
Engagement Tracking:

Tracks likes, shares, comments on each pulse
Monitors "insightful" vs "misinformed" reactions
Uses engagement data for relevance scoring
6. Industry Categorization
Each pulse is categorized by:

Industry (Technology, Healthcare, Finance, etc.)
Category (within industry - Marketing, Development, etc.)
Content Type (general, project, news, poll)
Target Audience (based on user analysis)
The system creates a dynamic, personalized professional feed that combines user-generated content with AI-powered industry insights, automatically adapting to community activity and individual preferences.
