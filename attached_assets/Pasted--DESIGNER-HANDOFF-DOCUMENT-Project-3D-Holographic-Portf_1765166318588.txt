Supports all user fields and structured data

🪄 6. INTERACTION ANIMATION SYSTEM
Global Animations
Interaction	Animation
Hover	Lift, glow, shadow
Scroll	Fade up, stagger
Click	Press down
Modal	Scale fade in
Particle system	Continuous float
Scanline	Loop, low opacity
Holographic gradient	Mouse tracked
Rules:

Easing: ease-out

Duration: 250-600ms

No janky bounce

Maintain performance
