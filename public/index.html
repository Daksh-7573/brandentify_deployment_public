📋 How to Use URL Input Feature

  1. Click on "URL Input Demonstration" above
  2. Select either "Images" or "Video" tab
  3. Enter your media URL in the input field
  4. Click "Add URL" or press Enter
  5. See live preview of your added media
  6. Remove URLs by clicking the × button
